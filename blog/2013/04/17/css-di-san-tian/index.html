Css 第三天

Apr 17th, 2013

中间间断了几天，今天开始继续学习 css。

今天主要研究了以下几块内容：

  • background 背景
  • text 文本
  • font 字体
  • list 列表
  • table 如何设置表格样式
  • outline 轮廓
  • 框模型概述

css 总体上要比 html 复杂许多，有更多的东西需要记忆。 有时候我们自己会懈怠，不想继续，但是，坚持往往是通向成功的唯一途径。

come on！ 丹丹
